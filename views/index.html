URL Shortener Microservice

Welcome to the URL Shortener Microservice!

Have you ever had the need to share some url in a short format?, so you can achieve this right now using this tool.

How to use?

  1. Fill the form above with the url that you want to short and click the button.
    http(s)://www.example.com(/more/routes) is the format admitted.

  2. You will receive a response like that
    {"original_url":"www.freecodecamp.org","short_url":1}
    where the value of short_url is your new shorted url

  3. Make a request in the web browser with the new shorted url
    Example: [project_url]/api/shorturl/1
    You have just been redirected to the original site link.
